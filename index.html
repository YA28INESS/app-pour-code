Apprendre les Langages

Découvre la différence entre Java et JavaScript :

Java

C'est quoi ? Un langage très puissant utilisé pour créer des applications de téléphone (Android), des logiciels d'ordinateur et de grands systèmes de serveurs.

Exemple : Le jeu Minecraft original a été entièrement codé en Java !

JavaScript

C'est quoi ? Le langage du web. Il sert à rendre les pages internet vivantes et interactives (comme les boutons sur lesquels tu cliques ou ce jeu !).

Exemple : C'est le langage qu'on utilise en ce moment même pour faire bouger la cible rouge.
